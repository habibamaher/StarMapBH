# StarMapBH
An interactive map created for the astronomy community in Bahrain showing night sky visibility in Bahrain, updated with satellite data and interesting facts

This project was created as part of a GitHub and project management lab for the Software Design & Engineering course and is inspired by the mission of the National Space Science Agency (NSSA) Bahrain to promote public interest in space science.

How it aids NSSA
- Inspires Curiousity in astronomy amongst the community in the Kingdom of Bahrain.
- Connects space science to local weather conditions which encourages observation of the stars in real time.
- Serves as a prototype, for future NSSA educational tools or mobile apps.

Technologies Used
- HTML, CSS3, Javascript.
- OpenWeatherMAP API - For cloud visibility and weather conditions.
- Stellarium API - For constellation and star mapping
- GitHub - version control and collaboration
